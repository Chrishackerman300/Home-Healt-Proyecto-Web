    Proyecto Web Home Health 

    Este es un proyecto mio, donde estare realizando una pagina desde cero sobre el diseño y la salud del hogar,
    esta web esta libre de elementos de derechos de autor, asi que si quieres descargar todos los elementos quehare en este 
    proyecto no tendras ninguna complicacion.

    Para este proyecto o la web es especifico estare haciendo diferentes paginas para que tenga una inteaccion e informacion 
    muy completa para el usuario. Donde podra navegar libremente en diferentes partes y secciones de la web.

    Para esto utilizare HTML para hacer el maqueteado y seccionamiento de la creacion de todo el cuerpo de nuestra web, despues para
    darle un buen diseño lo estare haciendo con CSS. 

    Estare utilizando JavaScript para darle interaccion a algunas secciones de la pagina web, principalmente en el formulario de registro
    e inicio de sesion para el usuario y para un sidebar menu.

    Todo esto es un proyecto sencillo pero completo para los web developers front-end.

    Estare utilizando Git para trabajar en diferentes ramas y no arriesgar la version inicial del proyecto en general y lo iremos comiteando 
    cada que me asegure de haber terminado con dicha seccion de la web.

    Al finalizar con este proyecto pondras muchas herramientas basicas y expertas tanto en HTML, CSS y JavaScript, y con el derecho de autor
    que es mio soy el programador Christian Mendoza.
